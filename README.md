## 初始化项目
```
	npx create-react-app wls-new
	cs wls-new
	npm start
```

### 组织目录结构
	src
		components
		service
		views
		App.css
		App.js
		index.js

## 安装路由
```
	npm install react-router-dom --save
```

## 集成sass
```
    npm install node-sass --save
```

## 集成element-ui
```
    npm i element-react --save
    npm install element-theme-default --save
    npm install react-hot-loader@next --save-dev
```
